The source code for this project is able to be compiled with JDK 17 from the terminal.

In order to run the program from the terminal, cd into the src folder wthin Lab4. Enter the commands:
(1) - javac SortingMain.java
(2) - javac ReadingData.java
(3) - javac QuickSorting.java
(4) - javac MyLinkedMerge.java
(5) - java SortingMain (input folder) (quickSort output folder) (mergeSort output folder)
   * input folder - "../(name)"
   * quickSort output folder - "../(name)"
   * mergeSort output folder - "../(name)"
